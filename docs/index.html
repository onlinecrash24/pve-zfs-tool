Features

Everything you need to manage ZFS on Proxmox VE

🗄

Pool & Dataset Management

Pool overview with health, IO stats, scrub control, upgrade detection. Datasets separated by type — filesystems and VM volumes with type-specific actions.

📸

Snapshot Management

Interactive timeline and table views. Create, clone, diff, rollback with automatic VM/LXC stop & restart. Cross-pool cloning via zfs send/recv.

🔄

File-Level Restore

Browse and restore files from LXC snapshots directly. VM restore via kpartx — supports ext4, xfs, btrfs, NTFS. BitLocker detection included.

🔍

Snapshot Check

Retention policy verification, gap detection, stale snapshot warnings, count mismatches (SOLL/IST), missing labels, and manual snapshot tracking.

🤖

AI Reports

Automated analysis via OpenAI, Anthropic, Ollama, or any compatible API. Scheduled reports, interactive chat, PDF export, raw data export.

🛡

Health & Monitoring

ARC stats, SMART disk health, ZFS events, ZDB deep diagnostics for degraded pools. Restore session cleanup with one-click bulk operations.

🔔

Notifications

Telegram, Gotify, and Matrix. Configurable per event — scrub, snapshot, rollback, pool errors, health warnings, host offline alerts.

🔒

Security

CSRF protection, input validation, rate limiting, secure sessions, security headers, path traversal prevention, SSH TOFU, reverse proxy ready.

🌐

Multi-Host SSH

Manage multiple Proxmox nodes via SSH. Auto-generated Ed25519 keys, connection testing, persistent known hosts with TOFU verification.
